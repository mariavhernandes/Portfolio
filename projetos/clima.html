Climas do Brasil - Aplicativo Climático

Aplicativo mobile desenvolvido para consultar informações climáticas de cidades brasileiras em tempo real por meio de uma API. O projeto também conta com suporte à internacionalização, permitindo a troca rápida de idioma.



Funcionalidades

  • Consulta da temperatura em cidades do Brasil
  • Internacionalização de idiomas
  • Layout simples e intuitivo


Tecnologias Utilizadas

  • Android Studio
  • Java
  • XML
  • API de clima
