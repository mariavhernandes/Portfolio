Calculadora Windows - Com Histórico

Aplicativo mobile desenvolvido no Android Studio para realizar operações matemáticas básicas. O principal diferencial do projeto é o histórico de cálculos, permitindo que o usuário visualize e acompanhe as operações realizadas durante a sessão.



Funcionalidades

  • Operações básicas: soma, subtração, multiplicação e divisão
  • Histórico de cálculos realizados
  • Layout limpo e intuitivo para dispositivos móveis
  • Opção para limpar o visor e o histórico


Tecnologias Utilizadas

  • Android Studio
  • Java
  • XML
